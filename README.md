# Environmental Health Observatory

## Project Overview

This project investigates how weather conditions and air quality change over time and how they may relate to environmental health risk.

## Research Question

How do heat and air quality patterns change over time, and how might they relate to environmental health risk?

## Data Sources

- Open-Meteo API
- EPA AirNow API (or OpenAQ)

## Tools

- Python
- Pandas
- SQLite
- Streamlit
- GitHub
